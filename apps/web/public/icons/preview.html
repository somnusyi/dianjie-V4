滇界 · App 图标 3 个方向

同一图标显示 180/100/60 三个尺寸 — 真机最小是 60px 启动器图标。手机上看效果：长按图标选"添加到主屏幕"用第一行的尺寸预览。

方案 A · 单字 monogram

柿色背景 + 米白「滇」字 + 弱化拼音底标

  • 最简洁、辨识度最高
  • 缩到 60px 仍清晰
  • 调性最像"专业工具"（非外卖 App）
  • 适合：直接上线，不出错

方案 B · 连锁山地

三家递增门店剪影 + 云南山地落日

  • 表达"连锁餐饮 + 云南"双重定位
  • 有故事、产品化
  • 缺点：小尺寸细节略多
  • 适合：愿意承担一点辨识成本换叙事

方案 C · 碗 + 云

深色底 + 柿色碗 + 米白云朵

  • 不靠汉字，国际化容易
  • 深色调显得高端
  • 缺点："餐饮" 表达比 A 弱
  • 适合：未来想拓展海外/英文版

说话告诉我"用 A / B / C"，我把它生成全套尺寸（iOS 1024 + Android 各种 dpi + 启动屏），然后跑 cap-assets generate 一键铺到原生工程。
